Project Description

This was my first Android app I ever made. It has a very basic layout that allows you to view all of your created notes and make new ones. I also added an option to delete all notes.

Current isses are that if the app crashes or you were to close the app without saving your current note, you will lose it. I want to add a auto save feature that will save the note every character update.
