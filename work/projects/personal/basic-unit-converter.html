Project Description

I made this app to help me learn how to use web based lanuages like HTML, CSS and JavaScript to quicky make mobile apps using Apache Cordova. This is a simple app that allows you to convert some common units of measurement to other related forms of measurement.

This project took me three days to complete. The hardest part was making the Units Of Imformation Calculator because it basically required a lot of repitition that I could not find an easy way to get round.
